## Description:

  * Pill reminder and alarm for all medication needs.
  * Family and caregiver support via “Medfriend” functionality.
  * Medicine tracker.
  * Refill reminders.
## The user can:
 * Add Medication
 * Add Health taker
 *Display the drug notification dialog when pressing on an item

  *Firebase
